# Useful Node.js Commands

---

# Check Version

```bash
node -v
```

---

# Check npm Version

```bash
npm -v
```

---

# Initialize Project

```bash
npm init
```

---

# Quick Initialization

```bash
npm init -y
```

---

# Install Dependency

```bash
npm install express
```

---

# Install Development Dependency

```bash
npm install --save-dev nodemon
```

---

# Install All Dependencies

```bash
npm install
```

---

# Remove Package

```bash
npm uninstall express
```

---

# Update Packages

```bash
npm update
```

---

# List Packages

```bash
npm list
```

---

# Run Script

```bash
npm run dev
```

---

# Start Application

```bash
npm start
```

---

# Cache Clean

```bash
npm cache clean --force
```

---

# Show Package Information

```bash
npm info express
```

---

# Install Global Package

```bash
npm install -g nodemon
```

---
