# Verify Installation

Check Node Version

```bash
node -v
```

Example

```
v22.18.0
```

---

Check npm Version

```bash
npm -v
```

---

# What is npm?

Node Package Manager.

Comes bundled with Node.js.

---

# Install Packages

```bash
npm install
```

---

# Install Single Package

```bash
npm install express
```

---

# Install as Development Dependency

```bash
npm install --save-dev nodemon
```

---

# Global Installation

```bash
npm install -g nodemon
```

---

# Check Installed Packages

```bash
npm list
```

---

# Uninstall Package

```bash
npm uninstall express
```

---

# Update Package

```bash
npm update
```
